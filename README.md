# 🌱 Agri-Gen

Agri-Gen is a modern web platform designed to serve both **B2B and B2C users in the agriculture ecosystem**. It provides a seamless digital experience for connecting stakeholders, managing agricultural services, and improving accessibility to agri-based solutions.

🔗 **Live Demo:** https://agrigen.vercel.app

---

## 🚀 Features

- 🌾 B2B & B2C support  
- 📊 Modern and responsive UI  
- ⚡ Fast performance with optimized frontend  
- 🔐 Scalable architecture  
- 🌐 Deployed and accessible online  

---

## 🛠️ Tech Stack

- **Frontend:** React / Next.js  
- **Styling:** Tailwind CSS  
- **Deployment:** Vercel  

---

## 📦 Installation & Setup

Clone the repository and install dependencies:

```bash
git clone https://github.com/yashchaudhari11/agri-gen-web.git
cd agri-gen-web
npm install
npm run dev
```

Open your browser and visit:

```bash
http://localhost:3000
```
